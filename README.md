# Homebrew Tap

Personal Homebrew tap for kluzzebass projects.

## Installation

```bash
brew tap kluzzebass/tap
```

## Available Formulas

### reflag

Translate command-line flags between different CLI tools.

```bash
brew install reflag
```

Homepage: https://github.com/kluzzebass/reflag

## Usage

After tapping this repository, you can install any formula with:

```bash
brew install <formula-name>
```
