# Homebrew Tap

Personal Homebrew tap for kluzzebass projects.

## Installation

```bash
brew tap kluzzebass/tap
```

## Available Formulas

### gastrolog

Log aggregation service

```bash
brew install gastrolog
```

Homepage: https://github.com/kluzzebass/gastrolog

### gqlt

A minimal, composable command-line client for GraphQL operations

```bash
brew install gqlt
```

Homepage: https://github.com/kluzzebass/gqlt

### lazyhue

TUI tool for managing Hue bridges

```bash
brew install lazyhue
```

Homepage: https://github.com/kluzzebass/lazyhue

### mcpipboy

MCP server providing AI agents with data generation and validation tools

```bash
brew install mcpipboy
```

Homepage: https://github.com/kluzzebass/mcpipboy

### reflag

Translate command-line flags between different CLI tools

```bash
brew install reflag
```

Homepage: https://github.com/kluzzebass/reflag

### shushtls

Local PKI: issue TLS certs for your services, install root CA on devices (home/lab)

```bash
brew install shushtls
```

Homepage: https://github.com/kluzzebass/shushtls
